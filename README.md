## 📘 Description

This is a minimal **NestJS** project scaffold that includes:

- A `UserController` for basic REST endpoints
- DTOs (`create-user.dto.ts`, `update-user.dto.ts`, `list-all-users.dto.ts`) for handling data
- Ready to expand into a full-featured backend API

---

## Project Setup

```bash
# Install dependencies
npm install

# Run in development mode
npm run start:dev

# Or run normally
npm run start
```
